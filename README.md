1.Project Title & Description

Expense Manager

Expense Manager is a simple web-based project that helps users manage their personal finances. The user can add balance, record expenses, and view the remaining balance. The project shows how expenses affect available money and helps users understand budgeting in a practical way. It is built using HTML, CSS, and JavaScript.


2.Problem Statement

Many people find it difficult to track their daily expenses and control their spending. Without proper tracking, users may overspend and lose awareness of their financial situation. This project provides a simple solution by allowing users to record expenses and automatically calculate the remaining balance.


3.Features Implemented

Add balance
Add expenses with amount, category, and note
Automatic calculation of total expenses
Automatic update of remaining balance
Expense list display
Input validation for incorrect values
Responsive layout for mobile and desktop


4.DOM Concepts Used

document.getElementById()
innerText and innerHTML
createElement()
appendChild()
Event handling using onclick
Dynamic updating of web page content


5.Steps to Run the Project

Open the project folder
Make sure index.html, style.css, and script.js are in the same folder
Open index.html in any web browser
Start using the application by adding balance and expenses


6.Known Limitations

Data is not saved after page refresh
Expenses cannot be edited or deleted
No user login system
No graphical analysis of expenses
